# Sistema de Controle de Manutenção (SCM) - InfoTech IFPB

## Descrição do Projeto
O Sistema de Controle de Manutenção (SCM) é um projeto acadêmico desenvolvido para a disciplina de Web Front-End.  
O projeto consiste em um site estático para a empresa **InfoTech IFPB**, especializada em manutenção de computadores.  

O objetivo é permitir que clientes vejam os serviços oferecidos e abram chamados de manutenção de forma prática e intuitiva.

## Estrutura do Projeto

O projeto está dividido em 5 partes:

- **Parte 1:** Protótipo estático (HTML sem CSS)
- **Parte 2:** Estilização básica com CSS
- **Parte 3:** Formulário funcional com validação HTML5
- **Parte 4:** Layout responsivo e cards de serviços
- **Parte 5:** Refinamento visual, transições e preparação para deploy

Cada parte contém:

- `index.html` → Página Home
- `servicos.html` → Página de Serviços
- `chamado.html` → Página de abertura de chamados
- `css/style.css` → Arquivo de estilos (a partir da Parte 2)
- `img/` → Imagens dos serviços (a partir da Parte 4)

## Funcionalidades

- Página Home com informações detalhadas da empresa
- Página de Serviços com cards detalhados de Hardware, Software e Redes
- Página de Chamado com formulário completo e validação HTML5
- Layout responsivo para diferentes tamanhos de tela
- Transições suaves em links e cards

## Tecnologias Utilizadas

- HTML5
- CSS3
- Flexbox e Grid
- Media Queries para responsividade

## Como acessar o site no GitHub Pages

1. O site está publicado no GitHub Pages.  
2. Acesse o link em qualquer navegador para explorar:
   https://4ndrein4.github.io/projetoscmfinal/
   - **Home:** informações sobre a empresa e missão.
   - **Serviços:** cards detalhados com Hardware, Software e Redes.
   - **Abrir Chamado:** formulário de manutenção.
4. Todos os arquivos (HTML, CSS, imagens) estão carregados automaticamente.
5. Não é necessário instalar nada localmente para visualizar o site.

## Observações

- Imagens usadas nos cards estão em `img/`.
- O formulário não envia dados para servidor, pois é um protótipo estático.
