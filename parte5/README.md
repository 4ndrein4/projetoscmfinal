# Sistema de Controle de Manutenção (SCM)

Projeto desenvolvido para a disciplina de desenvolvimento web do IFPB.

## Descrição
Site estático para a empresa fictícia InfoTech IFPB, permitindo
visualização de serviços e abertura de chamados de manutenção.

## Tecnologias Utilizadas
- HTML5
- CSS3
- Flexbox
- Grid Layout

## Funcionalidades
- Página institucional
- Página de serviços com cards
- Formulário de abertura de chamados
- Layout responsivo

## Publicação
Projeto preparado para publicação no GitHub Pages.
